# 图片网站

这是一个二次元动漫图片展示网站，使用墨天逸随机图片API获取图片数据。

## 功能特点

- **图片展示**：展示100张二次元动漫图片
- **分批加载**：优化加载速度，避免一次性加载过多图片
- **去重功能**：确保每张图片都是唯一的
- **错误处理**：当图片加载失败时显示占位符图片
- **图片放大**：点击图片可全屏查看大图
- **图片下载**：支持下载当前查看的图片

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 网站会自动加载100张二次元动漫图片
3. 点击任意图片卡片，会打开全屏模态框显示大图
4. 在模态框中点击"下载图片"按钮，即可下载当前图片
5. 点击关闭按钮或模态框外部，即可关闭模态框

## 技术实现

- **前端技术**：HTML5, CSS3, JavaScript
- **图片API**：墨天逸随机图片API (https://api.mtyqx.cn/)
- **占位符API**：文本转图片API (https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image)
- **加载优化**：分批加载、图片懒加载
- **错误处理**：图片加载失败处理、重复图片去重

## 注意事项

- 本网站使用的是第三方图片API，图片版权归原作者所有
- 由于API限制，可能会出现部分图片加载失败的情况，网站会自动显示占位符图片
- 网站支持响应式设计，适配不同屏幕尺寸
